## ShiftLefter

Open-source test framework and quality toolkit — quality as a dimension, not a phase.

**→ [Main repository](https://github.com/shift-lefter/shiftlefter)** · [Website](http://shiftlefter.com)

ShiftLefter is a Cucumber-compatible BDD framework built in Clojure that adds vocabulary definition and enforcement, compile-time macro expansion, multi-actor contexts, and structured event output on top of the full Gherkin spec.

v0.3.x · 900+ tests · actively developed

Built by [Gabriel Wilkins](https://www.linkedin.com/in/Gabriel-Wilkins).
